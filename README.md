# PopX Mobile UI

A pixel-perfect mobile UI application built with React and Vite. Features a 375px mobile layout centered on desktop with four pages: Welcome, Register, Login, and Account.This project was developed as a React JS qualifier assignment demonstrating clean component architecture, client-side routing, and structured UI implementation.

## Tech Stack

- **React 19** — Functional components with hooks
- **Vite** — Build tool and dev server
- **React Router DOM** — Client-side routing
- **Plain CSS** — No frameworks or libraries

## Features

- Mobile-first 375px container centered on desktop
- User registration with localStorage persistence
- Login with credential validation
- Account profile page with avatar
- Reusable Button and InputField components
- Clean, professional UI with Rubik font

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build

```bash
npm run build
npm run preview
```
